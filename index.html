Trailmate

User Friendly

I have focused on re-designing the main information, and making that more accessible to mobile users.

The application opens up into a map spanning the entire screen. The user will be prompted with the option to:

"Look for tracks near by"
 alternatively, the user can browse the map or type key words into the Search Bar

Tracks appear as colored lines, and loop tracks are shaded in to represent a closed path.

Color represent grade of difficulty, and red means a track is CLOSED.

When a track is selected it will glow, and a box containing information about the trail will pop up from the bottom of the screen.
